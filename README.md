# Slasher music bot

Slasher is a python based discord music bot. 
It requires:
- python 3.10 or above
- discord.py 1.7.3
- PyNacl 1.4.0
- discord-py-interactions 3.0.2
- async-timeout 3.0.1

## Commands:

- /play (play music from youtube)
- /skip
- /pause
- /resume
- /stop
- /leave
- /ping
- /settings announce songs
- /settings shuffle play
- /np (now playing)
- /queue
- /clear all
- /clear duplicates
